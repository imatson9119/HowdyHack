Submitting Events

Event submission is the backbone of Hungry Ags. Currently, we allow any registered Google account to submit events. Consequently, some general guidelines are required in order to maintain quality submissions. These can be observed below:
  • Obscene or fowl language is not to be used when submitting events. All submissions should be suitable for any audience.
  • All events submitted must contain free food.
  • No personal information may be shared.
Violation of any one of these guidelines can result in an immidate termination of your account. If you have any questions, feel free to contact hungryagsofficial@gmail.com


Tips on submitting your events:
  • Keep your description concise- a long excessive paragraph can lose the intrest of readers!
  • Including the type of food your event will serve can be helpful to those using the filtering tools.
  • If you use a valid Texas A&M Building code (HRBB, MSC, ect.) somewhere in your event location, a map icon will display on your event to allow users to easily view it on a map.
  • If you make a mistake when submitting your event or the event is cancelled, clicking on one of your own events will display the options to either edit or delete the event.
